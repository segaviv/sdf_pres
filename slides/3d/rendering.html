Rendering Techniques

Lights

Normals can be calculated numerically via the (combined) sdf.

\[ N(p)=\nabla f(p)=\begin{pmatrix} \frac{f(p + \{h, 0, 0\}) - f(p)}{h} \\ \frac{f(p + \{0, h, 0\}) - f(p)}{h} \\ \frac{f(p + \{0, 0, h\}) - f(p)}{h} \end{pmatrix} \]

And can be used to calculate lighting

\[ I(p) = Material(p) light\_color \left( N(p) \cdot light\_dir \right) \]
